还需要等待下载相应的gradle配置

在 Gradle 中配置依赖仓库，可以通过在 `build.gradle` 文件中添加以下内容：

repositories {
    mavenCentral() // 使用 Maven 中央仓库
    google() // 使用 Google Maven 仓库
    jcenter() // 使用 JCenter 仓库
    // 可以添加其他自定义的仓库，例如：
    // maven { url "https://my-repository.com/maven" }
}
